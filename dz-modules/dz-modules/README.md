# DZ Modules Workspace

This workspace contains the production modularization of Discovery Zero v2:

- `dz-hypergraph`: hypergraph, graph persistence, inference adapters, tools, and skills
- `dz-verify`: claim extraction and verification pipeline
- `dz-engine`: MCTS core engine and orchestration
- `dz-mcp`: MCP server for agent integration

Source code is copied from `/personal/Discovery-Zero-v2` into this workspace and
kept behavior-compatible, with import path refactors only.
